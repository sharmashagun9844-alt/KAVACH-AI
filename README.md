# KAVACH-AI
An advanced, privacy-first cybersecurity companion built with Python and Streamlit for the Rakshak Hackathon. Simulates Whisper+LLM multi-modal audio scam telemetry, auto-generates structured cybercrime complaints, and integrates interactive compliance mapping for India's DPDP Act 2023 to protect users from social engineering.

# Project Overview
KAVACH-AI v3 is an end-to-end, privacy-focused cybersecurity application built for the Rakshak Hackathon.

The system directly tackles the exponential rise in phone-based financial fraud and complex social engineering tactics. Engineered using a Python backend and an interactive Streamlit architecture wrapped in a custom "Fintech Midnight Blue & Muted Gold" UI, KAVACH-AI v3 provides real-time risk telemetry, automated legal redressal drafts, and proactive digital rights awareness for citizens to combat scams safely.

# 🚀 Core Architecture & Features
1. Multi-Modal Audio Scam Telemetry (Simulated Pipeline)
Ingestion & Processing: Simulates an asynchronous pipeline combining neural speech-to-text (Whisper-style) and contextual LLM inference engine logic to analyse threat vectors.

Psychological Trigger Extraction: Scans incoming context for psychological manipulation tactics, including extreme urgency, panic-inducing language, fake law enforcement impersonation, or fraudulent banking requests.

Dynamic Visual Risk Mapping: Leverages Plotly Graph Objects to project real-time threat scores onto an interactive, color-coded visual risk dial.

2. End-to-End Automated Redressal
Dynamic Complaint Drafting: Once a threat is identified, the system's core logic automatically compiles session metadata into a structured, formal legal complaint layout.

# Submission-Ready Outputs: 
Generates standardized text output customized perfectly for immediate filing on the official National Cyber Crime Reporting Portal.

Persistent Emergency Access: Includes a sticky, high-visibility taskbar dedicated to India's 24x7 National Cyber Fraud Helpline (1930) under the Ministry of Home Affairs.

3. Strict Compliance Framework (DPDP Act, 2023)
Consent Gatekeeping: No data analysis can execute without an explicit, session-state tracked affirmative consent prompt, mapping directly to modern compliance standards.

Active Knowledge Architecture: Features a dedicated reference module displaying user rights under the Digital Personal Data Protection Act, highlighting:

Section 6: Withdrawal of Consent

Section 11: Right to Know / Access Information

Section 12: Right to Correction & Erasure

Section 13: Grievance Redressal Mechanisms

Section 66: Visual warnings regarding the statutory ₹250 Crore penalty metrics for data processors.

4. Conversational Awareness Bot
An on-demand, sliding UI element containing a specialized chatbot payload designed to educate vulnerable users against ongoing vector attacks like Aadhaar fraud, spoofing, and lottery scams.

# 💻 Technical Implementation Details
Language: Python 3.x

Framework: Streamlit (leveraging custom HTML/CSS injections for deep interface control)

Visualization: Plotly (Dynamic Gauge Telemetry)

State Control: Strict st.session_state management to prevent processing leakage prior to data-owner consent.
